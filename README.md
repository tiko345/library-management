#Library Management System
A console-based library management system written in C++ that allows users to manage books - 
borrow, return, add, remove them, also save or load the info to or from a file


##Project Structure
library
├──include/
│  ├──book.h
│  └──library.h
├──src/
│  ├──book.cpp
│  ├──library.cpp
│  └──main.cpp
└──README.md


##features
- add, remove and display books
- borrow and return books
- search by Id, title or author
- save and load data from a file
